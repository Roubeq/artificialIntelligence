# Классификатор интентов на основе Random Forest

Модель машинного обучения для классификации текстовых запросов по интентам с использованием TF-IDF и Random Forest.

## Описание

Проект представляет собой систему классификации текстовых запросов (интентов) на русском языке. Модель обучается на предварительно размеченном датасете и может использоваться для определения намерения пользователя в чат-ботах, голосовых ассистентах и других системах обработки естественного языка.

**Архитектура решения**:
- Векторизация текста выполняется с помощью **TF-IDF** (TfidfVectorizer) с настройками: 5000 наиболее значимых признаков, использование униграмм и биграмм (1-2 граммы), фильтрация слишком редких и слишком частых терминов
- В качестве классификатора применяется **Random Forest** — ансамблевый метод, объединяющий множество решающих деревьев. Гиперпараметры модели подобраны для баланса между качеством и обобщающей способностью: 200 деревьев, максимальная глубина 20, минимальное количество образцов для разделения узла — 5, балансировка весов классов для работы с несбалансированными данными

**Предобработка текста**:
- Поддерживается два режима: с использованием библиотеки **spaCy** (при наличии модели `ru_core_news_sm`) и упрощенный regex-вариант
- При наличии spaCy выполняется лемматизация, удаление стоп-слов и знаков препинания, оставляются только значимые токены длиной более 2 символов
- В упрощенном режиме текст приводится к нижнему регистру, удаляется пунктуация и лишние пробелы

**Обучение и валидация**:
- Датасет загружается из CSV-файла `intent_dataset_final.csv` с колонками `text` и `intent`
- Данные разделяются на обучающую (80%) и тестовую (20%) выборки со стратификацией для сохранения распределения интентов
- После обучения модель выводит точность на тестовой выборке и полный отчет о классификации
- Обученная модель и векторизатор сохраняются в формате pickle (joblib) в директорию `random_forest_model/`

**Структура проекта**:
- `SKLearn.py` — основной скрипт обучения модели: загрузка данных, предобработка, обучение Random Forest, сохранение результатов
- `intent_dataset_final.csv` — размеченный датасет для обучения (должен быть создан предварительно)
- `random_forest_model/` — директория для сохранения обученной модели и векторизатора

**Особенности реализации**:
- Автоматическое определение доступности spaCy с fallback на упрощенную обработку
- Использование `class_weight='balanced'` для работы с несбалансированными классами
- Параллельные вычисления (`n_jobs=-1`) для ускорения обучения
- Возможность тестирования модели на произвольных примерах после обучения

Такой подход позволяет создать легковесный и эффективный классификатор, который может быть легко интегрирован в чат-бота для определения намерений пользователя без использования тяжелых нейросетевых моделей.
