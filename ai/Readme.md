# Консольный чат-бот на Python

Консольный чат-бот с распознаванием естественного языка, интеграцией с API погоды, хранением данных в SQLite и сбором статистики по пользователям.

## Описание

Проект представляет собой консольного чат-бота, построенного на основе регулярных выражений для сопоставления пользовательских запросов с заранее определёнными паттернами. При запуске бот инициализирует базу данных SQLite, содержащую три таблицы: `users` (пользователи), `chat_log` (история диалогов) и `weather_queries` (запросы погоды).

Ядро бота реализовано в классе `ChatBot` (модуль `bot_core.py`), который управляет циклом обработки сообщений, передачей управления обработчикам из `patterns.py` и логированием взаимодействий. Каждый обработчик привязан к регулярному выражению и выполняет конкретную функцию: приветствие, запрос времени или даты, предсказание в стиле «шара судьбы» и т.д.

Особенностью проекта является интеграция с внешним API погоды (Weatherstack) — при запросе погоды бот обращается к сервису, парсит ответ и возвращает пользователю актуальные данные о температуре, влажности, скорости ветра и описании погоды. Все успешные запросы погоды сохраняются в базу данных с привязкой к пользователю.

Бот поддерживает идентификацию пользователей по имени: после того как пользователь представится, его данные сохраняются в таблице `users`, а последующие сообщения привязываются к его ID. Это позволяет вести персональную статистику — количество сообщений, дату первого и последнего визита, количество запросов погоды. Для просмотра статистики предусмотрена команда «моя статистика».

Архитектура проекта разделена на модули по принципу единственной ответственности:
- `main.py` — точка входа, запуск и обработка критических ошибок;
- `bot_core.py` — основной класс бота, управление диалогом и логикой;
- `patterns.py` — регулярные выражения и функции-обработчики команд;
- `database.py` — работа с SQLite: инициализация, сохранение пользователей, логирование сообщений, запись погодных запросов, получение статистики;
- `weather_api.py` — взаимодействие с API погоды Weatherstack;
- `logger.py` — дополнительное логирование в текстовый файл (оставлено для обратной совместимости).
