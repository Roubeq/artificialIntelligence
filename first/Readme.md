# Консольный чат-бот на Python

Простой консольный чат-бот с поддержкой базовых команд, логированием диалогов и функцией предсказаний.

## Описание

Чат-бот реализован на Python с использованием регулярных выражений для распознавания пользовательских запросов. При запуске бот загружает набор паттернов из модуля `patterns.py`, каждый из которых связан с функцией-обработчиком. Пользовательский ввод обрабатывается в цикле: сообщение проверяется на соответствие одному из паттернов, после чего вызывается соответствующий обработчик. Если совпадений не найдено, бот возвращает случайный ответ из списка стандартных фраз.

Для логирования диалогов используется отдельный модуль `logger.py`, который записывает каждое сообщение пользователя и ответ бота в текстовый файл с временной меткой. Бот способен запоминать имя пользователя в рамках текущей сессии благодаря хранению состояния в экземпляре класса.

Проект состоит из четырёх модулей:
- `bot.py` — точка входа, запуск и обработка критических ошибок;
- `handlers.py` — основной класс `ChatBot`, управляющий логикой обработки сообщений;
- `patterns.py` — регулярные выражения и функции-обработчики команд;
- `logger.py` — функция для записи диалогов в лог-файл.
